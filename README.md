## ‚ú® Features

* **100% Responsive:** Full mobile-first design that adapts seamlessly to desktop, tablet, and mobile screens.
* **Multi-Page Site:** Includes three distinct, fully-styled pages:
    * **Home/Products:** Main product showcase with a filterable grid.
    * **About:** Brand story, mission, and feature cards.
    * **Contact:** A complete contact form and info section.
* **Dynamic Product Filtering:** JavaScript-powered filtering on the home page to sort products by category (All, Earbuds, Smartwatches, etc.).
* **Interactive Mobile Menu:** A smooth, animated "hamburger" menu for easy navigation on small devices.
* **Smooth Animations:** Subtle hover effects, card lifts, and fade-in animations on scroll to create a premium, modern feel.
* **Pure HTML/CSS/JS:** Built from scratch with no external CSS or JS frameworks (like Bootstrap or React) to demonstrate core web development skills.

This project was built using:

## üèÅ How to Run Locally

To get a local copy up and running, follow these simple steps.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/aziz-ch52/soundverse.git](https://github.com/aziz-ch52/soundverse.git)
    ```

2.  **Navigate to the project folder:**
